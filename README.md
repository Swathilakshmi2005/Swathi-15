**Student CRUD Application**

A simple Student Management System built using ASP.NET Core Web API, SQLite, and Next.js.  
This project demonstrates full CRUD operations with a clean frontend and RESTful backend.

## Features

- Create student records  
- View all students  
- Update student details  
- Delete student records  
- REST API using ASP.NET Core  
- SQLite database integration  
- Next.js frontend with Axios  


## Technologies Used

### Backend
- ASP.NET Core 8 Web API  
- Entity Framework Core  
- SQLite  
- Swagger (Swashbuckle)

### Frontend
- Next.js (React)  
- TypeScript  
- Axios  
- HTML & CSS  

---

## Database

- **SQLite**
- Lightweight file-based database
- No separate server required
